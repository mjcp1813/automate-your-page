The World Wide Web

The World Wide Web or the Web consist of computers that use the Internet to communicate with one another. The basics of the Web include a computer with a browser installed, the Internet, servers and HTTP. HTTP is the main protocol of the web, the Internet can be consider the highway that connects the computers together. the servers are computer that are design to store an distribute information.

Files found on the World Wide Web can be any of the following types:
  • Plain Text
  • HTML
  • Images
  • Videos
  • Music

Hypertext Markup language

Hypertext Markup Language, HTML for Short, which makes up most of the content on the web. HTML is made up of text, symbols and code that tells the browser how to display the page Content is what you see Markup...What it looks like referenced to other documents.....Images and Videos Links ....To other pages There was discussion in the makeup of an HTML Documents including the contents in the header using the tag head as well as makeup of the body of the document using the tag body.

HTML Attributes

HTML attributes provide additional information about an HTML element. They consist of a name and a value. An example of a attribute would be the href attribute. Where href would be the name and a URL link would be the value

Inline Vs Block

A block element such as p for paragraph or div for division creates and imaginary block around whatever is between the opening and closing tag. This imaginary block will expand it with and height automatically as item such as test are added within the opening and closing tag. Inline elements usually do not start a new line and are used to change some item in the line like a bold tag.

Structured Document with HTML

Browsers build a tree like structures of the HTML page, the tree like structure is more like a flow chart that branches out rather than a real tree, The HTML language controls the structure of the page This tree tells the browser what is text, images.

Boxes, Boxes and More Boxes

All of the content on a web page can be broken down into boxes of all various sizes. These boxes are defined by div tags. Using boxes to build web pages allows those boxes to have their own position on the page as their own style.

CSS and Styling

Whereas the CSS language (Cascading Style Sheet) controls the style section of the document. That includes the visual aspect of the page including font type, font color, background color and many other attributes that determine the visual makeup of the page.

Python

Python is a programing language that is characterized by its readability and it's easy to use syntax structure. It is considered a high level programing language.

Variables and Strings

Variables utilize an assignment statement which allows the programmer to name an expression and refer to that name instead of having to input the expression each time it is needed. A string is any character on a keyboard that can be displayed on the screen. The character and can be alpha, numeric or characters. Strings are declared by encasing them in either single, double or triple quotation marks. A string may not have any characters at all, an empty string.

Functions

Using functions in programing equates to mapping inputs to outputs. Defining a function whereby giving the function a name and an input (one or more) allows the programer to to multipalte the output by changing the value of the input without having to develope a new function

Loops: if & while

A while loop permits code to be excuted repetitively based on a specific condition. A while loop consist of an expression and a statement. As long as the expression is true the loop continues, once a false is returned the program moves to the next line in the code An if loop
